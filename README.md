# CosmoCats — проект

Коротко: CosmoCats — небольшой веб-приложение-чат с образным AI (кот). Проект написан на Flask. Эта версия добавляет кнопку на титульной странице, которая показывает случайное изображение кота, получаемое с публичного endpoint TheCatAPI (без ключа).

Что сделано в этом коммите:
- Добавлена функция `get_random_cat()` в `ai_core.py` — использует `requests` для получения случайного URL изображения кота.
- Добавлен маршрут `/random-cat` в `app.py`, который возвращает JSON `{ "url": "..." }`.
- Обновлён `templates/index.html` — титульная карточка и кнопка для показа кота.
- Обновлён `static/script.js` — кликабельная кнопка и отображение изображения.

Как запустить
1. Создайте виртуальное окружение и установите зависимости (Windows PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Запустите приложение:

```powershell
setx FLASK_APP app.py
python app.py
```

Откройте http://127.0.0.1:5000/ — на главной странице нажмите «Показать случайного кота».

Примечания и ограничения
- TheCatAPI свободно доступен для простого поиска изображений без обязательного API-ключа; однако частые запросы могут быть ограничены.
- AI-ядро (`ai_core.py`) использует локальную модель через `transformers` и `torch`. Если модель не установлена, генерация ответов потребует загрузки и может быть медленной.
- Перевод и ответы ИИ настроены для русского языка. При пустом или невозможном ответе ИИ вернёт: `у меня лапки, я не могу это сделать :(`

Если будут пожелания по дизайну титульной страницы — опишите желаемый стиль, и я внесу правки в CSS/шаблон.
